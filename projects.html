DOM As the Source of Truth

Here are some projects that are included in this repository. Some of which were actually included in the talk.

From presentation:

  • Chart JS - A Stimulus controller for interacting with chart.js.
  • Form Progress - A Stimulus controller for showing the progress on filling out a form.
  • Table Filter - A Stimulus controller for filtering an HTML table.
  • Table Sort - A Stimulus controller for sorting an HTML table.

Other projects:

  • Check All - A Stimulus controlller to check all checkboxes.
  • Cloner - A Stimulus controlller for cloning elements.
  • Courier - A Stimulus controlller for copying data between elements.
  • Eraser - A Stimulus controlller for conditionally clearing the data from elements.
  • Force Check - A simple JavaScript library to force a checkbox to stay checked.
  • Highlighter - A Stimulus controller for highlighting words.
  • History - A Stimulus controller for working with the browser history.
  • Select Input - A Stimulus controller to make a searchable select box.
  • Show/Hide - A Stimulus controller to conditionally show/hide elements.
